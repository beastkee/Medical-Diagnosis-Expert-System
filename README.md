# Medical-Diagnosis-Expert-System

A medical diagnosis expert system built as a self-contained web application.

## Live Demo

View the live page here: **[https://beastkee.github.io/Medical-Diagnosis-Expert-System/](https://beastkee.github.io/Medical-Diagnosis-Expert-System/)**

## Repository Structure

```
Medical-Diagnosis-Expert-System/
├── index.html              # Main HTML page (entry point for GitHub Pages)
├── css/
│   └── styles.css          # All styles: layout, components, responsive breakpoints
└── js/
    ├── knowledge-base.js   # Data: KNOWLEDGE_BASE (14 diseases) + SYMPTOM_GROUPS
    ├── inference-engine.js # Logic: weighted symptom-matching inference engine
    └── app.js              # UI: rendering, symptom selection, step navigation
```

## Usage

Open `index.html` in any modern web browser, or visit the live GitHub Pages link above.  
The application runs entirely in the browser — no server or installation required.
